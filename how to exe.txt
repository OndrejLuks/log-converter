HOW TO MAKE PYTHON SCRIPTS PACKAGED INTO .EXE

	- Using CX_FREEZE
	
	- INSTALATION
		- natively in Python from 3.3 and higher
		- pip install --upgrade cx_Freeze
		
	- PREPARATION
		- if using multiple processes -> multiprocessing.freeze_support()
		
	- EXECUTING
		- COMMAND LINE
			- cxfreeze -c hello.py --target-dir dist
		
		- BUILD_SETUP.PY
			- better
			- more options
			
	- CREATING A BATCH FILE
		- python build_setup.py build
